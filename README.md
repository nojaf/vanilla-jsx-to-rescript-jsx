# Vanilla JSX to ReScript JSX

See [online tool](https://vanilla-jsx-to-rescript-jsx.web.app)!

## Contributing

Install dependencies

```shell
bun i
```

Run tests

```shell
bun run tests
```

Format code

```shell
bun run fmt
```

Launch tool

```shell
bun run tool
```

## Deploying tool

This thing is deployed to Firebase.

```shell
bun run tool:build
firebase deploy --only hosting
```
